# Products API

The Products API is auto-generated by Supabase from the `products` table. Use it to fetch, create, update, and delete products in the catalogue.

## Base URL
```
https://your-project.supabase.co/rest/v1
```

## Authentication

All requests require two headers:
```bash
apikey: YOUR_SUPABASE_ANON_KEY
Authorization: Bearer YOUR_SUPABASE_ANON_KEY
```

### GET /products

Returns all products in the catalogue.

**Request**
```bash
curl https://your-project.supabase.co/rest/v1/products \
  -H "apikey: YOUR_KEY" \
  -H "Authorization: Bearer YOUR_KEY"
```

**Response**
```json
[
  {
    "id": "uuid",
    "name": "White Linen Shirt",
    "category": "top",
    "image_url": "https://storage.url/shirt.png",
    "price": 1299,
    "in_stock": true
  }
]
```

### GET /products?category=eq.top

Returns products filtered by category.

**Categories:** `top`, `bottom`, `footwear`, `accessory`

### POST /products

Adds a new product to the catalogue.

**Request body**
```json
{
  "name": "Black Trousers",
  "category": "bottom",
  "image_url": "https://storage.url/trousers.png",
  "price": 2499,
  "in_stock": true
}
```

## Error codes

| Code | Meaning |
|---|---|
| `200` | Success |
| `400` | Bad request — check your request body |
| `401` | Unauthorized — check your API key |
| `404` | Resource not found |
